# PlayOS
Web技术实现的应用层面操作系统。

## Feature
- 最新Web技术试验田，我们承诺，永不做跨浏览器兼容
- 纯前端实现的应用级别操作系统，功能可热插拔
- 基于Polymer的Web Component标准开发模式
- 采用Pure.css布局，简单干净有个性
- 图标采用iconfont.cn平台支持
- 牛逼的应用级插件扩展
